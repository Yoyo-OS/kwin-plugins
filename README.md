# YoyoOS KWin Plugins

[![Build](https://github.com/Yoyo-OS/kwin-plugins/actions/workflows/build.yml/badge.svg)](https://github.com/Yoyo-OS/kwin-plugins/actions/workflows/build.yml)

## Dependencies

`sudo pacman -S extra-cmake-modules qt5-base qt5-declarative kconfig kdecoration kguiaddons kcoreaddons kconfigwidgets kwindowsystem kwayland kwin`

## Build

```
mkdir build
cd build
cmake ..
make
sudo make install
```
